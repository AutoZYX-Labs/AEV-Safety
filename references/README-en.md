# References

[中文](README.md)

This directory stores the reference seed for AEV Safety.

| File | Description |
|---|---|
| `AEV-Safety-main-references.bib` | BibTeX seed covering standards, papers, guidance documents and public page leads |

## Use Boundary

- The BibTeX file is a reference starting point for the manuscript and review, not final submission formatting.
- Every standard entry should be rechecked against the official status page and authorized text before formal citation.
- Official status pages can support standard names, versions, status and scope. Clause-level interpretation requires authorized standard text.
- Company websites, press releases, videos and job pages may anchor scenarios, but they do not prove safety maturity, certification status or long-term field performance.

## Update Rule

New references should include DOI, official page URL, access date, evidence level and use boundary whenever possible. Materials that cannot be verified should stay in internal notes before entering the public BibTeX file.
