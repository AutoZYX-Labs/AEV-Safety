# 参考文献说明

[English](README-en.md)

本目录保存 AEV Safety 的参考文献种子文件。

| 文件 | 说明 |
|---|---|
| `AEV-Safety-main-references.bib` | BibTeX 参考文献种子，包含标准、论文、指南和公开页面线索 |

## 使用边界

- BibTeX 文件是论文和综述的参考文献起点，不是最终投稿格式。
- 每个标准条目应在正式引用前重新核对官方状态页和授权文本。
- 官方状态页可支持标准名称、版本、状态和范围；条款级解释必须依赖授权标准全文。
- 企业官网、新闻稿、视频和招聘页可作为场景线索，不作为安全成熟度、认证状态或长期运行表现证据。

## 更新规则

新增参考文献时应尽量提供 DOI、官方页面 URL、访问日期、证据等级和使用边界。无法核验的材料应先放入内部笔记，不直接进入公开 BibTeX。
