# AEV Safety 评价矩阵 v1.0

[English](evaluation-matrix-en.md)

## 定位

本矩阵定义自主工程车辆安全评价应记录的指标族。它不是固定阈值表。不同机器、场地、任务和监管环境需要校准阈值。公开仓库先给出指标结构、记录字段和证据用途。

## 六类后果

| 后果类别 | 说明 | 典型指标 |
|---|---|---|
| 人员安全 | 工人、驾驶员、维护人员和旁观者暴露 | 最近距离、侵入次数、急停触发、人员识别漏检 |
| 资产与基础设施 | 车辆、工具、固定设施、卡车、料斗、输送带损伤 | 碰撞、接触力、资产损伤、误入禁区 |
| 生产连续性 | 工期、节拍、停机和恢复 | 平均修复时间（Mean Time To Repair, MTTR）、任务中断次数、恢复时间、人工介入率 |
| 任务质量 | 满载率、卸料准确性、开沟深度、桩位、平整度 | 偏差、重工率、撒料率、质量不合格率 |
| 环境与资源 | 能耗、撒漏、粉尘、能源补给、材料浪费 | 能耗/任务、物料损失、环境异常 |
| 恢复能力 | 故障后能否安全停止、撤离、恢复或维修 | 最小风险状态（Minimal Risk Condition, MRC）成功率、恢复路径、监督响应、现场处置时间 |

## 指标族

### 1. 场地边界指标

- 作业场地与任务设计条件（Worksite and Task Design Conditions, WTDC）完整度：是否声明场地类型、地形、物料、工具、人员区、禁入区、通信、定位和监督条件。
- 地图新鲜度：场地地图、料堆边界、卸料点、障碍物和禁入区更新频率。
- 基础设施健康：通信质量、定位质量、调度系统、急停系统、围挡和警示系统状态。

### 2. 感知与语义指标

- 人员/设备/料堆/车厢/坑槽识别召回率和误检率。
- 低置信度、分布外（Out-of-Distribution, OOD）输入、遮挡、粉尘、夜间、雨雪、强反光等触发条件记录。
- 感知失败到安全降级的时间。

### 3. 移动与路径指标

- 路径跟踪误差：按任务容差报告，不只报告道路横向误差。
- 速度、加速度、铰接角、转弯半径、倒车距离和最小安全距离。
- V 型循环路径长度、循环时间和路径冲突次数。

### 4. 工作装置与物料指标

- 铲斗/工具姿态误差、入料角、卸料高度、工具速度、力/压力限制。
- 满载率、撒料率、卸料偏差、开沟深度偏差、桩位偏差、压实质量。
- 工具异常、液压冲击、打滑、卡料、料堆坍塌和任务失败记录。

### 5. 监督与恢复指标

- 远程监督请求率、响应时间、确认质量和误确认率。
- 自动降级触发、MRC 执行、人工接管、现场处置和恢复时间。
- 控制权转移记录：自动、远程监督、远程驾驶、现场人工、维护。

### 6. 安全案例指标

- 每个安全主张是否有证据、假设、限制和失效样本。
- 每个测试通过准则是否连接到 WTDC、动态移动与作业任务（Dynamic Moving and Working Task, DMWT）和后果类别。
- 每个运行异常是否能回填到场景分类、触发条件、功能不足和缓解措施。

## 测试矩阵模板

| 字段 | 说明 |
|---|---|
| machine_type | 装载机、挖掘机、矿卡、打桩机、开沟机、清舱机器人等 |
| worksite_type | 矿山、拌合站、港口、施工现场、光伏场地、沙盘、月面类测试场 |
| task_type | 铲掘、装载、卸料、运输、打桩、开沟、压实、清舱、恢复 |
| WTDC | 地形、物料、天气、通信、定位、监督、人员区、禁入区 |
| DMWT | 移动动作、工作装置动作、任务质量动作、恢复动作 |
| trigger_condition | 触发条件，如粉尘、遮挡、低附着、通信退化、定位漂移 |
| insufficiency | 功能不足，如料堆识别不足、路径跟踪不足、卸料点判断不足 |
| consequence | 六类后果之一或多个 |
| mitigation | 减速、急停、降级、远程确认、重新规划、人工接管 |
| evidence | 仿真、封闭场地、实车、运行日志、人工评审 |

## 使用边界

本矩阵不声明任何通用安全阈值。阈值必须由具体机器、具体场地、具体任务、授权标准、试验数据和利益相关方共识确定。
