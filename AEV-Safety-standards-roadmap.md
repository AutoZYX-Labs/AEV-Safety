# AEV Safety Standards Roadmap

## Purpose

This roadmap organizes the standards work for autonomous engineering-vehicle safety. It is not a claim that all standards have been obtained or that clause-level mapping is complete.

## Current Standards Landscape

### Road-Vehicle Donor Standards

- ISO 26262: functional safety.
- ISO 21448: SOTIF, Safety of the Intended Functionality.
- ISO/SAE 21434: cybersecurity engineering.
- ISO 34502: scenario-based safety evaluation framework.
- UL 4600: autonomous product safety argumentation.

### Engineering and Mining Machinery Standards

Relevant families include earth-moving machinery safety, machine-control-system safety, mining autonomous guidance, off-highway dump-truck standards and collision warning standards. Current public status information is useful for positioning, but clause-level mapping requires authorized text or authorized catalogue access.

### Adjacent Mobile Work Standards

ISO 18497-1:2024 is relevant because it addresses partially automated, semi-autonomous and autonomous agricultural machinery and tractors. It is not an engineering-vehicle standard, but it shows how automated mobile work outside public roads is entering formal standardization language.

### Robot SOTIF Signal

The national-standard project on robot SOTIF entered public consultation on 2026-06-02, with consultation ending on 2026-07-02 according to the public project page. It is an adjacent standardization signal for mobile physical AI. It must not be treated as a published engineering-vehicle requirement.

## Proposed Standardization Topics

1. Vocabulary for mobile working machine systems.
2. Worksite and task design conditions.
3. Dynamic moving and working task.
4. SOTIF-oriented risk-chain analysis for engineering-vehicle tasks.
5. WTDC-DMWT scenario testing matrix.
6. Remote supervision and operational monitoring.
7. Evidence classes for public, internal, test and field evidence.
8. Live hazard log and post-change revalidation.
