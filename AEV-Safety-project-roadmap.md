# AEV Safety Project Roadmap

## Positioning

AEV Safety is a long-term project on autonomous engineering-vehicle safety. The project is not only for writing a paper. It is intended to support deeper research, standardization discussion, testing and evaluation, conference communication, and product-grounded engineering practice.

The project scope covers:

1. Wheel loaders in batching plants, ship holds, ports, steel plants and semi-closed yards.
2. Excavators and attachments for trenching, crushing, grabbing, solar construction and pipeline construction.
3. Autonomous mining haulage and unmanned transport systems.
4. Solar piling and other repeatable construction tasks.
5. Future lunar, Mars and extreme-environment mobile work.

## Core Research Questions

1. How should the safety object shift from a road vehicle item to a mobile working machine system?
2. How should ODD be rewritten as worksite and task design conditions?
3. How should DDT be rewritten as dynamic moving and working task?
4. How should SOTIF risk chains include work-equipment hazards and task consequences?
5. How can road-vehicle safety methods help without forcing road-traffic assumptions onto engineering work?
6. What evidence is needed before a system can be discussed as mature, deployable or ready for standardization?
