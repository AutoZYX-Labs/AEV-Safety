# Process Archive / 过程归档

本文件夹用于本地保存 AEV Safety 项目的过程材料，包括早期草稿、证据卡、模板、渲染文件和内部背景 source extracts。

This folder keeps local process materials for the AEV Safety project, including earlier drafts, evidence cards, templates, rendered files, and private background source extracts.

这些材料不进入公开 GitHub 仓库。公开仓库只保留论文稿、分类体系、参考架构、评价矩阵、标准版图、材料吸收记录、参考文献、图件和项目网页。

These materials are intentionally not tracked in the public GitHub repository. The public repository keeps the manuscript, taxonomy, reference architecture, evaluation matrix, standards landscape, source synthesis, bibliography, figures and project website.

本地归档结构：

Local archive layout:

- `20260604-process/`: prior drafts, tables, process templates, evidence logs, rendered outputs, and figure/reference source files.
- `source-extracts/`: internal background source extracts, including Li Xuefei's business-plan PDF and the Move the Earth and Beyond deck.
